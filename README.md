🌌 DreamVault

A creative playground to learn and practice HTML, CSS, and JavaScript by building real projects.

⸻

🚀 About the Project

DreamVault is a beginner-friendly web development project designed to help you learn by building.
Instead of just studying theory, this project focuses on creating something visually appealing and interactive using core frontend technologies.

Whether you’re just starting out or refining your basics, DreamVault acts as your learning vault of experiments, designs, and ideas.

⸻

🎯 Goals
	•	💡 Learn HTML structure and semantic elements
	•	🎨 Master CSS styling, layouts, and responsiveness
	•	⚡ Understand JavaScript fundamentals & interactivity
	•	🛠️ Build real-world mini features and UI components
	•	📈 Improve practical development skills through hands-on coding

⸻

🧰 Tech Stack
	•	HTML5 – Structure
	•	CSS3 – Styling & Layout
	•	JavaScript (Vanilla) – Logic & Interactivity

⸻

✨ Features
	•	🌙 Clean and modern UI design
	•	📱 Responsive layout (mobile-friendly)
	•	⚡ Interactive elements using JavaScript
	•	🎨 Styled components and sections
	•	🔧 Beginner-friendly code structure
